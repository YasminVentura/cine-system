# Sistema de Cinema

Este projeto foi desenvolvido como parte avaliativa da disciplina de Programação 3, utilizando Java e os conceitos de orientação a objetos. O objetivo era aplicar os conhecimentos aprendidos em sala de aula em um projeto com tema livre, escolhido pelo aluno

## Tema

Sistema de compra de ingressos para sessões de cinema

## Funcionalidades

- Listagem de filmes disponíveis
- Escolha de sessão e assento
- Compra de diferentes tipos de ingresso (inteiro, estudante, VIP)
- Tratamento de exceções para entradas inválidas e assentos ocupados
